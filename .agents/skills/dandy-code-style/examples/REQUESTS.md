# Request Examples

```text
Apply dandy-code-style to this diff.
```

Route to the smallest matching child set after reading the diff.

```text
Use dandy-code-style to clean up this controller.
```

Likely route: `naming-clarity`, `control-flow-simplifier`, and `php-error-handling` if failures are involved.

```text
Review this AI-generated patch before I commit it.
```

Route to `ai-code-ownership`, then add `testability-maintenance` if tests are missing.

```text
Clean up comments and TODOs in this module.
```

Route to `comment-curator`.

```text
Make this project easier for a new developer to start.
```

Route to `repo-onboarding`.
